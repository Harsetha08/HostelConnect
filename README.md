# HostelConnect
To improve communication between management and students about complaints and improve efficiency.

---

## Features
- complaints about (missing,food,plumbing,carpentry,electrical,cleaning & others)
- hostel facilities
- hostel contacts
- hostel food menu
- hostel blueprint
- hostel rules
- hostel updates

---

## TechStack
- Frontend- ReactJS
- Backend - Java Springboot
- Database- MySQL

---

## Role
- Student
- Wardern
- CareTaker
- Management
- Admin

---

## UI/UX Design

### Public

1. Home

- What HostelConnect is
- How it helps students & management
- Quick links (Raise Complaint, Menu, Facilities)

2. Facilities

- Hostel facilities (rooms, water, power, gym, playground)
- Timings & availability

3. Login / Sign In

- Role-based login
- Redirects to dashboard

### Student

1. Food Menu

- Daily / weekly mess menu
- Special notes (holidays, changes)

2. Hostel Rules

- Entry/exit rules
- Discipline & safety rules

3. Hostel Blueprint

- Hostel layout image
- Blocks, floors, rooms info

4. Contacts
- Warden
- Caretaker

### Office & emergency contacts

5. Student Dashboard

- Complaint summary
- Latest hostel updates
- Quick actions

6. Raise Complaint

- Category-based complaint form
- Priority selection
- Submit & track

7. My Complaints

- Complaint history
- Status tracking
- Responses from staff

### Warden/Caretaker Pages

1. Dashboard

- Assigned complaints
- Priority-based list
- Overview of hostel issues
- Pending & critical complaints

2. Work Update Page

- Change complaint status
- Add remarks

3. Complaint Assignment

- Assign complaints to caretakers
- Escalate to management

### Management Pages

1. Management Dashboard

- Complaint analytics
- Facility performance
- Reports

### Admin Pages

1. Admin Dashboard

- Overall system health

2. User & Role Management

- Add / edit users
